# 项目说明 — 股票市场行业轮动与因子收益分析

大数据可视化课程期末项目。

---

## 项目简介

分析 2022–2025 年中国 A 股市场，通过四层数据处理流水线生成 12 张交互式 Plotly 图表，探索行业轮动、宏观经济周期、因子收益与决策支持。采用 TradingView/Bloomberg 暗色终端风格。

## 运行环境

- Python 3.11+
- 操作系统：Windows / macOS / Linux 均可

## 数据获取（Tushare Pro）

1. 注册 Tushare Pro 账号：<https://tushare.pro>
2. 获取 API token，设置环境变量或在 `数据获取.py` 中配置
3. 运行：`python 数据获取.py`（从 `数据集/` 目录）

该脚本会下载 2022–2025 年全部 A 股日线、日基本面、指数成分股等数据。

## 数据处理流水线

按顺序执行，CSV 是层间唯一数据交换格式：

```bash
cd 源代码/

# 第1层: 行业聚合
python build_industry_data.py          → datasets/processed/01_industry/ (5 CSV)

# 第2层: 宏观环境
python build_macro_cycle.py            → datasets/processed/02_macro/   (2 CSV)

# 第3层: 因子归因（含因子收益率计算）
python build_factor_attribution.py     → datasets/processed/03_factor/  (3 CSV)

# 第4层: 决策支持
python build_decision_support.py       → datasets/processed/04_decision/ (3 CSV + signals/)
```

若数据已有预处理结果，可直接运行可视化脚本。

## 可视化生成

```bash
cd 源代码/

python -m visualizations               # 生成 12 张 HTML + PNG → 可视化成果/
python build_dashboard.py              # 生成综合仪表盘 → 可视化成果/index.html
```

查看结果：用浏览器打开 `可视化成果/index.html`（综合仪表盘，推荐），或单独打开 `可视化成果/` 下任意 HTML 文件。

```
    ├──→ 图2(风格)        ├──→ 图4(气泡)          │
    │                      │                        │
    └──→ 图9(箱线图)      ├──→ 图5(网络) ──群落──→ 图6(平行坐标)
                           │                        │

                                    │                    │

                          图12(预警) ←── signals/ ←── 图1-11
```
