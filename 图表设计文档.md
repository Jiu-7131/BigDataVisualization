# 十二张图制作要求与数据需求总览

---

## 图1：宏观经济周期仪表盘

| 维度 | 说明 |
|:---|:---|
| **图表类型** | 仪表盘 + 周期判定条 |
| **核心功能** | 综合判定当前经济周期阶段，输出配置方向 |
| **布局结构** | 上半区2×2仪表盘（PMI/CPI/国债/社融），下半区1×4周期阶段条 |
| **视觉要求** | 每个仪表盘为半圆弧，红-黄-绿颜色分区，指针指向当前值；当前周期阶段高亮+箭头标注 |
| **关键标注** | 当前值、环比变化、迷你12月趋势线、关键分项注释 |
| **交互要求** | 悬停看详情、月份选择器切换历史、点击周期阶段展开行业建议 |
| **数据来源** | `cn_pmi`(pmi010000)、`cn_cpi`(nt_val)、`cn_bond`(y10)、`cn_social`(社融存量同比) |
| **更新频率** | 月度 |
| **输出参数** | 周期阶段标签（复苏/过热/滞胀/衰退）传递给所有下游图表 |

## 图2：市场风格轮动热力三角图

| 维度 | 说明 |
|:---|:---|
| **图表类型** | 三元相图 |
| **核心功能** | 展示大小盘/价值成长风格漂移轨迹 |
| **布局结构** | 三角坐标系为主体，右上角图例，底部时间轴 |
| **视觉要求** | 三个顶点标注"大盘价值""大盘成长""小盘"，数据点按年份着色连线，当前时点用大号高亮圆点 |
| **关键标注** | 箭头标注近期漂移方向（如"→向小盘迁移"），标注经济周期背景色分区 |
| **交互要求** | 悬停显示日期+三种风格收益率、拖动时间轴回看历史轨迹 |
| **数据来源** | `index_daily`：沪深300价值(000919.SH)、沪深300成长(000918.SH)、中证1000(000852.SH)的日收益率 |
| **更新频率** | 月度 |
| **输出参数** | 当前风格偏向标签，与图1周期阶段叠加 |

## 图3：日历热力图

| 维度 | 说明 |
|:---|:---|
| **图表类型** | 日历热力图 |
| **核心功能** | 展示行业月度收益率排名时序，发现季节性规律 |
| **布局结构** | 行=行业，列=年月，颜色=排名分位，右侧行业名称，顶部宏观周期色带 |
| **视觉要求** | 红=排名前（跑赢）、蓝=排名后（跑输）、白=中等；行业按聚类排序，相似轮动模式的行业排在一起 |
| **关键标注** | 连胜标记（连续N月前5）、顶部叠加PMI折线作为宏观参照 |
| **交互要求** | 点击单元格弹出该月收益分布箱线图、切换"月度排名"/"超额收益"双模式 |
| **数据来源** | `daily`(个股日线) + `index_member_all`(成分股)，自建行业月度收益率排名 |
| **更新频率** | 月度 |
| **输出参数** | 行业月度胜率矩阵 → 传递给图10桑基图 |

---

## 图4：动态气泡图

| 维度 | 说明 |
|:---|:---|
| **图表类型** | 动态气泡图 |
| **核心功能** | 展示行业风险收益定位与跨周期迁移 |
| **布局结构** | X轴=收益率，Y轴=波动率，气泡大小=市值，四象限分割线 |
| **视觉要求** | 气泡颜色=行业分组（周期/成长/防御），当前年份气泡有轮廓描边，历史年份半透明，选中行业显示轨迹连线 |
| **关键标注** | 四象限标签（I进攻/II理想/III防御/IV陷阱），标注异常离群点的事件说明 |
| **交互要求** | 年份滑块/自动播放、点击行业显示迁移轨迹、按经济周期阶段绘制凸包 |
| **数据来源** | `daily`(计算年收益率、年化波动率) + `daily_basic`(total_mv市值) + `index_member_all` |
| **更新频率** | 年度/半年度 |
| **输出参数** | 行业风险收益分类标签 |

---

## 图5：行业相关性网络图

| 维度 | 说明 |
|:---|:---|
| **图表类型** | 力导向网络图 |
| **核心功能** | 展示行业间关联结构与抱团/分化现象 |
| **布局结构** | 节点=行业，连线=相关性>阈值，节点大小=市值，节点颜色=收益率 |
| **视觉要求** | 正相关蓝线、负相关红线、线粗=相关强度；行业群落自动聚类用虚线框圈出；红涨绿跌着色节点 |
| **关键标注** | 标注"桥梁行业"（连接多个群落的行业），标注群落名称 |
| **交互要求** | 时间滑块切换、双击行业高亮其关联网络、点击群落展开成员列表 |
| **数据来源** | `daily`(计算行业日收益率序列) + `index_member_all`，滚动窗口计算行业间相关系数矩阵 |
| **更新频率** | 月度 |
| **输出参数** | 行业群落标签 → 传递给图6平行坐标图做分组着色 |

---

## 图6：平行坐标图

| 维度 | 说明 |
|:---|:---|
| **图表类型** | 平行坐标图 |
| **核心功能** | 展示各行业在多因子维度上的得分轮廓 |
| **布局结构** | 6-8根平行纵轴（因子），每条折线=一个行业，右侧因子重要性柱状图 |
| **视觉要求** | 线条颜色按图5的行业群落分组，当前选中行业高亮加粗、其余灰色半透明，每轴标注得分方向箭头 |
| **关键标注** | 因子名称+得分方向（如"估值→越便宜越靠右"），轴顶/轴底标注高低含义 |
| **交互要求** | 悬停行业线查看因子值、轴上拖拽刷选筛选行业、时间切片器切换时点、双时点叠加对比 |
| **数据来源** | `daily`(计算动量/波动率/换手率因子) + `daily_basic`(PE/PB/ROE/市值因子)，需行业中性化处理 |
| **更新频率** | 周度/月度 |
| **输出参数** | 行业多因子得分表 → 传递给图7瀑布图 |

---

## 图7：因子收益贡献瀑布图

| 维度 | 说明 |
|:---|:---|
| **图表类型** | 瀑布图 |
| **核心功能** | 将行业超额收益拆解为各因子的贡献 |
| **布局结构** | 横轴从左到右：市场平均→估值贡献→动量贡献→质量贡献→波动率贡献→特质收益→行业实际收益 |
| **视觉要求** | 正向贡献绿色上升柱、负向贡献红色下降柱、起点/终点蓝色，每个柱子标注贡献值 |
| **关键标注** | 柱子顶部标注"+X%"或"-X%"，底部标注该因子当期IC值 |
| **交互要求** | 下拉菜单切换行业、点击某因子贡献柱跳转到图6高亮该因子轴 |
| **数据来源** | 基于图6的因子暴露 + 量化研究员C计算的因子收益率序列，收益归因 = 因子暴露 × 因子收益率 |
| **更新频率** | 月度 |
| **输出参数** | 各因子边际贡献 → 反馈给图8拥挤度判断 |

---

## 图8：因子拥挤度监控仪表

| 维度 | 说明 |
|:---|:---|
| **图表类型** | 仪表盘组合 |
| **核心功能** | 监控因子失效风险，预警拥挤交易 |
| **布局结构** | 2×2排列4个速度表（估值/动量/质量/低波动），底部拥挤度历史折线 |
| **视觉要求** | 每个仪表盘绿-黄-红三区背景，指针指向当前拥挤度，旁注"安全/警戒/拥挤"标签，当前IC小字标注 |
| **关键标注** | 若进入红色拥挤区，仪表盘外框红色闪烁；标注上次进入该区域的时间和后续表现 |
| **交互要求** | 点击仪表盘展开该因子的拥挤度历史走势图 |
| **数据来源** | `moneyflow`(资金流向) + `daily_basic`(持仓集中度) + 因子多空组合累计收益偏离度（量化研究员C计算） |
| **更新频率** | 周度 |
| **输出参数** | 因子拥挤预警信号 → 传递给图12综合预警 |

## 图9：收益率箱线图

| 维度 | 说明 |
|:---|:---|
| **图表类型** | 分组箱线图/小提琴图 |
| **核心功能** | 对比牛熊市及经济周期各阶段下行业表现差异 |
| **布局结构** | X轴=行业，每个行业并列4个箱线图（复苏/过热/滞胀/衰退），或按牛熊震荡三分组 |
| **视觉要求** | 四阶段分别用不同颜色（与图1对应），箱体显示Q1-Q3，须线1.5IQR，离群点单独标注，叠加小提琴图显示分布形态 |
| **关键标注** | 标注牛熊非对称比（牛市均值-熊市均值）、标注每阶段跑赢概率 |
| **交互要求** | 点击行业切换为该行业的详细分布对比、支持周期阶段 vs 牛熊二分两种分组模式切换 |
| **数据来源** | `daily`(行业月度收益率) + 图1输出的周期阶段标签 + 牛熊市划分规则（万得全A涨跌20%判定） |
| **更新频率** | 季度 |
| **输出参数** | 行业周期敏感性指标（牛市贝塔/熊市贝塔/非对称比） |

## 图10：行业配置建议桑基图

| 维度 | 说明 |
|:---|:---|
| **图表类型** | 桑基图 |
| **核心功能** | 将"宏观环境→风格偏好→行业推荐"的决策链可视化 |
| **布局结构** | 三层流向：第一层=经济周期阶段，第二层=配置策略（防御/均衡/进攻），第三层=具体推荐行业 |
| **视觉要求** | 流量宽度=推荐权重，当前所处路径高亮加粗，其他路径半透明，当前阶段节点用图1颜色标注 |
| **关键标注** | 每层节点旁标注"胜率X%"，推荐行业旁标注建议权重 |
| **交互要求** | 悬停路径显示历史胜率、点击推荐行业弹出因子得分卡、支持手动调整周期阶段看不同推荐 |
| **数据来源** | 图1周期阶段 + 图3行业胜率矩阵 + 量化研究员D的配置规则引擎 |
| **更新频率** | 月度 |
| **输出参数** | 最终行业配置权重向量 |

## 图11：策略压力测试热图

| 维度 | 说明 |
|:---|:---|
| **图表类型** | 热力图 |
| **核心功能** | 模拟极端情景下各行业潜在回撤 |
| **布局结构** | 行=申万一级行业，列=6种压力情景，颜色=预期回撤幅度 |
| **视觉要求** | 绿=小回撤(<5%)、黄=中回撤(5-15%)、红=大回撤(>15%)，每个单元格标注具体数字，右侧汇总列（综合压力得分） |
| **关键标注** | 标记每个行业的最脆弱情景（最大回撤所在格加粗边框），底部标注各情景定义和幅度 |
| **交互要求** | 点击单元格弹出历史相似情景真实回撤对比、支持自定义压力幅度 |
| **数据来源** | `daily`(历史极端区间行业收益率) + 量化研究员D计算的压力情景回归系数 |
| **更新频率** | 季度 |
| **输出参数** | 行业压力脆弱性排名 |

## 图12：实时监控预警仪表盘

| 维度 | 说明 |
|:---|:---|
| **图表类型** | 多卡片网格仪表盘 |
| **核心功能** | 多维度异常信号综合监控 |
| **布局结构** | 3行×2列卡片：行业轮动信号、因子失效预警、拥挤交易监控、风格切换预警、流动性异常、尾部风险指标 |
| **视觉要求** | 每卡片顶部状态灯（绿/黄/红），主体=迷你折线图，下方一句话预警提示；红色预警卡片有边框闪烁 |
| **关键标注** | 预警卡片标注"触发时间"和"历史相似情况下的后续走势" |
| **交互要求** | 点击预警卡片跳转到对应详细图表（如拥挤预警→图8，风格预警→图2），支持预警阈值自定义 |
| **数据来源** | 汇总图1-图11的关键输出信号，设定阈值触发规则 |
| **更新频率** | 每日/每周 |
| **输出参数** | 综合预警信号列表 → 推送通知/邮件 |
