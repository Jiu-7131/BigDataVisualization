## 图1：宏观经济周期仪表盘

| 维度 | 说明                                                                      |
|:---|:------------------------------------------------------------------------|
| **图表类型** | 仪表盘 + 周期判定条                                                             |
| **核心功能** | 综合判定当前经济周期阶段，输出配置方向                                                     |
| **成功标准** | 仪表盘指针一眼定位当前周期阶段                                                         |
| **布局结构** | 上半区1×4仪表盘（PMI/CPI/国债/社融），下半区1×4周期阶段条                                    |
| **视觉要求** | 每个仪表盘为半圆弧，红-黄-绿颜色分区，指针指向当前值；当前周期阶段高亮+箭头标注                               |
| **关键标注** | 当前值、环比变化、迷你12月趋势线、关键分项注释                                                |
| **交互要求** | 悬停看详情、月份选择器切换历史                                                         |
| **数据来源** | `cn_pmi`(pmi010000)、`cn_cpi`(nt_val)、`cn_bond`(y10)、`cn_social`(社融存量同比) |
| **更新频率** | 月度                                                                      |
| **输出参数** | 周期阶段标签（复苏/过热/滞胀/衰退）传递给所有下游图表                                            |

### 周期判定算法

```
经济方向 ← PMI 的 12 月移动均线环比（上行/下行）
通胀方向 ← CPI 同比环比（上升/下降）

复苏 = PMI↑ + CPI↓    过热 = PMI↑ + CPI↑
滞胀 = PMI↓ + CPI↑    衰退 = PMI↓ + CPI↓

社融存量同比 → 确认经济方向（同步验证）
10Y国债收益率  → 确认通胀预期（交叉验证）
```

当社融或国债收益率与 PMI/CPI 信号矛盾时，输出置信度降级。

**输出 JSON 结构**（所有下游图表依赖）：
```python
{
    "phase": "复苏",          # 复苏/过热/滞胀/衰退
    "confidence": 0.85,       # 0~1，信号冲突时降低
    "drivers": {"pmi": "up", "cpi": "down",
                "social_financing": "up", "bond_yield": "down"}
}
```

## 图2：市场风格轮动热力三角图（修正为四象限散点图）

| 维度 | 说明 |
|:---|:---|
| **图表类型** | 四象限散点图（大小盘溢价 × 价值成长溢价） |
| **核心功能** | 展示大小盘/价值成长风格漂移轨迹 |
| **成功标准** | 散点轨迹 + 方向标注一眼看出风格漂移方向 |
| **布局结构** | 四象限散点图为主体，右上角图例，底部时间轴 |
| **视觉要求** | x轴=大小盘溢价（小盘−大盘），y轴=价值成长溢价（价值−成长）；散点按风格象限着色（大盘价值/小盘成长/大盘成长/小盘价值），散点大小=风格漂移强度，当前时点大号高亮圆点 |
| **关键标注** | 箭头标注近期漂移方向（如"→向小盘迁移"），四象限标签 |
| **交互要求** | 悬停显示日期+三种风格收益率 |
| **数据来源** | `index_daily`：沪深300价值(000919.SH)、沪深300成长(000918.SH)、中证1000(000852.SH)的日收益率 |
| **更新频率** | 月度 |
| **输出参数** | 当前风格偏向标签，与图1周期阶段叠加 |

**归一化方法**：不适用三元相图（三个指数收益率是独立变量，和不为100%）。改用2D四象限：
- x = 小盘累计收益 − 大盘累计收益（12月滚动）
- y = 价值累计收益 − 成长累计收益（12月滚动）

## 图7：因子收益贡献瀑布图

| 维度 | 说明 |
|:---|:---|
| **图表类型** | 瀑布图 |
| **核心功能** | 将行业超额收益拆解为各因子的贡献 |
| **成功标准** | 绿柱（正向贡献）最高者一眼看出主要收益来源 |
| **布局结构** | 横轴从左到右：市场平均→估值贡献→动量贡献→质量贡献→波动率贡献→特质收益→行业实际收益 |
| **视觉要求** | 正向贡献绿色上升柱、负向贡献红色下降柱、起点/终点蓝色，每个柱子标注贡献值 |
| **关键标注** | 柱子顶部标注"+X%"或"-X%"，底部标注该因子当期IC值 |
| **交互要求** | 下拉菜单切换行业 |
| **数据来源** | 基于图6的因子暴露 + 第3层计算的多空组合因子收益率序列（写入 `processed/03_factor/factor_returns.csv`），收益归因 = 因子暴露 × 因子收益率 |
| **更新频率** | 月度 |
| **输出参数** | 各因子边际贡献 → 反馈给图8拥挤度判断 |

### 因子收益率计算方法（第3层 `build_factor_attribution.py`）

每月末：
1. 对全市场个股做 PE/PB/动量/波动率/换手率/市值排序
2. 做多前 30%、做空后 30%（多空组合）
3. 多空组合下月收益 = 该因子当月收益率

写入 `processed/03_factor/factor_returns.csv`，图6读它做因子暴露校正，图7读它做收益归因。

## 综合仪表盘设计：「数据画廊」

### 定位

课程报告展示入口，非实时交易终端。强调叙事引导与工作量呈现。

### 设计方向

全暗沉浸式体验，每张图表作为独立"展品"展示，配分析注释卡片说明该图的分析意义和关键发现。

### 配色体系

| 层级 | 色值 | 用途 |
|:---|:---|:---|
| 画布底色 | `#0a0e17` | 页面最外层背景 |
| 卡片底色 | `#111827` | 信息卡片、注释卡片 |
| 强调色 | `#f59e0b` (琥珀金) | 进度指示器、活跃元素、关键标注 |
| 分割线 | `rgba(255,255,255,0.06)` | 元素间分隔 |
| 主文字 | `#e0e4eb` | 标题、导航 |
| 次要文字 | `#787b86` | 辅助说明、图表注释 |
| 图表主题 | `#0f1923` / `#1a2332` | 保持现有 config.py 暗色不变 |

### 字体

- 英文/数字标题：**Outfit**（几何无衬线，干净利落，适合数据展示）
- 中文正文：**Noto Serif SC**（衬线，学术报告质感）

两者均通过 Google Fonts 加载，提供 fallback 链。

### 布局结构

├────────────────────────────────────────────────────────┤

### 交互与动画

### 分析注释内容（每张图表一段）

| 图表 | 注释内容 |
|:---|:---|
| 图1 | 通过PMI、CPI、国债收益率、社融四个核心指标综合判定当前所处的经济周期阶段（复苏/过热/滞胀/衰退），为后续行业配置提供宏观背景 |
| 图2 | 追踪大小盘风格（沪深300价值/成长 vs 中证1000）的漂移轨迹，判断当前市场偏好大盘还是小盘、价值还是成长 |
| 图3 | 以日历热力图展示31个申万一级行业月度收益率排名，识别行业轮动规律和连胜/连败模式 |
| 图4 | 以气泡图动态展示各行业的风险-收益特征变化，观察行业在四象限（进攻/理想/防御/陷阱）之间的跨周期迁移 |
| 图5 | 通过网络图揭示行业间的关联结构，识别抱团现象和"桥梁行业"，以及行业群落划分 |
| 图6 | 用平行坐标展示各行业在估值、动量、质量、波动率等多因子维度上的综合得分轮廓 |
| 图7 | 将行业超额收益拆解为各因子的贡献，用瀑布图直观展示收益来源结构 |
| 图8 | 监控四大因子的拥挤程度，预警因子失效风险（进入红色区表示过度拥挤） |
| 图9 | 对比不同经济周期阶段下各行业的收益率分布，识别各阶段的优势行业和防御型行业 |
| 图10 | 桑基图展示从"宏观环境→风格偏好→行业推荐"的完整决策链，输出配置建议 |
| 图11 | 模拟利率飙升、通胀冲击等5-6种极端情景下各行业的预期回撤，评估行业脆弱性 |
| 图12 | 汇总前11张图表的关键信号，在单个仪表盘上集中监控各项预警指标 |

## 改动范围界定

**不可动**（已验证、已提交）：
