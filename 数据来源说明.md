# 数据来源说明

## 数据源

**[Tushare Pro](https://tushare.pro)** — 国内最大的免费金融数据开放平台，提供全面的 A 股市场数据接口。

## 获取数据接口一览

| 接口 | 说明 | 数据量 |
|:---|:---|:---:|
| `daily` | A 股日线行情 (OHLCV) | ~5,400 只 |
| `daily_basic` | 每日基本面 (PE/PB/市值等) | ~5,400 只 |
| `adj_factor` | 复权因子 | ~5,400 只 |
| `index_member_all` | 申万行业指数成分股 | 31 个行业 |
| `index_daily` | 风格指数日线 (沪深300价值/成长、中证1000) | 3 个指数 |
| `index_classify` | 申万行业分类 | 31 个行业 |
| `moneyflow` | 个股资金流向 | ~5,400 只 |
| `cn_pmi` | 中国制造业 PMI | 月度 |
| `cn_cpi` | 中国居民消费价格指数 | 月度 |
| `cn_bond` | 国债收益率 (10年期) | 日度 |
| `cn_social` | 社会融资规模存量同比 | 月度 |
| `cn_m` | 中国货币供应量 (M2) | 月度 |
| `trade_cal` | 交易日历 | 全部 |

## 时间范围

2022 年 1 月 — 2025 年 12 月（共 48 个月）

## 原始数据规模

- 个股日线数据：约 5,463 只股票 × 约 974 个交易日 ≈ 500 万条记录
- 个股基本面：同上规模
- 总计原始 CSV 文件：约 27,300 个

## 预处理后关键字段

| 字段 | 说明 |
|:---|:---|
| `ts_code` | 股票代码（如 `000001.SZ`）|
| `trade_date` | 交易日期 (`YYYYMMDD`) |
| `open` / `high` / `low` / `close` | 开/高/低/收盘价 |
| `vol` | 成交量（手）|
| `pct_chg` | 涨跌幅 (%) |
| `pe_ttm` | 市盈率 TTM |
| `pb` | 市净率 |
| `total_mv` | 总市值（万元）|
| `turnover_rate` | 换手率 (%) |
| `l1_industry` | 申万一级行业分类 |

## 宏观数据字段

| 字段 | 说明 |
|:---|:---|
| `pmi` | 制造业 PMI 指数 |
| `cpi_yoy` | CPI 同比涨幅 (%) |
| `m2_yoy` | M2 同比增速 (%) |
| `cycle_phase` | 美林时钟周期判定（复苏/过热/滞胀/衰退）|

## 数据预处理说明

1. **缺失值处理**：对个别交易日的缺失 PE/PB 使用前向填充
2. **异常值检测**：剔除涨跌幅超过 ±11% 的异常记录（A 股涨跌停限制）
3. **行业映射**：通过申万指数成分股文件将个股映射到 31 个一级行业
4. **月度聚合**：将日线数据聚合为月度收益率（几何累乘）
5. **因子构建**：每月按市值/PB/动量五分位分组，Top − Bottom = 因子收益率

## 数据可靠性

- Tushare Pro 数据与 Wind / Choice 等商用终端一致
- 所有原始 CSV 文件保留在 `tushare_all_interfaces/` 目录
- 数据处理全过程可复现
