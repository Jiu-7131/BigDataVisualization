股票市场行业轮动与因子收益分析

基于 A 股市场 2022–2025 年数据的多维度可视化研究
课程名称：大数据可视化
选题编号：选题3 — 股票市场行业轮动与因子收益分析
数据来源：Tushare Pro 金融数据开放平台
可视化工具：Python Plotly + Kaleido
完成日期：2026年6月

项目背景与问题定义

1.1 选题动机

中国A股市场拥有超过5,000家上市公司，覆盖31个申万一级行业，总市值逾80万亿元。不同行业在不同宏观经济周期、不同市场风格下的表现存在显著差异。例如，2022年全球加息周期中，能源和银行等价值型行业表现突出，而科技成长型行业大幅回撤；进入2024–2025年，AI产业浪潮又推动电子、计算机等科技行业领涨市场。投资者面临的核心问题是：如何在不同的宏观环境和市场风格下，科学地选择行业配置方向？

传统行业研究多依赖于基本面分析（如财务指标比较）或单一维度的技术指标（如相对强弱）。这种方法忽略了两类关键信息：一是宏观经济周期对行业轮动的系统性驱动（如美林时钟框架）；二是因子层面的量化证据（如规模因子、价值因子、动量因子对不同行业超额收益的解释力）。

本项目的核心思路是：构建一个"宏观环境 → 行业结构 → 因子归因 → 决策支持"四层递进分析框架，通过对A股市场2022–2025年全样本数据的系统处理和交互式可视化，直观揭示行业轮动的内在规律，辅助投资决策。

1.2 分析目标

  1. 识别宏观周期与行业轮动的关系：基于美林时钟框架，划分复苏、过热、滞胀、衰退四个经济周期阶段，观察各阶段中哪些行业系统性跑赢市场。
  2. 量化风格因子对行业收益的解释力：构建规模因子（SMB）、价值因子（HML）、动量因子（MOM），分析各行业在不同因子上的暴露程度及其对超额收益的贡献。
  3. 评估因子拥挤风险：通过估值价差、因子内相关性、换手率集中度等指标，监控因子拥挤状态，预警潜在的因子崩溃风险。
  4. 构建可视化决策仪表盘：将上述分析整合为12张交互式图表，并汇总为单一仪表盘页面，支持快速切换、键盘导航和搜索过滤。

数据描述与预处理

2.1 数据来源

本项目的全部数据来自Tushare Pro（https://tushare.pro），是国内使用最广泛的免费金融数据开放平台。数据覆盖2022年1月至2025年12月共48个月的交易历史，涉及约5,400只A股股票。

2.2 数据清洗过程

（1）缺失值处理：个股基本面数据中，部分交易日缺失PE/PB（如新股上市初期、ST股票停牌期间）。对于月度聚合场景，取月末最后一个有效值；对于无法填充的极端缺失，在因子构建时跳过该股票。

（2）异常值检测：剔除单日涨跌幅超过±11%的异常记录（A股正常涨跌停板为±10%，科创板/创业板±20%，个别极端值可能来自数据错误）。剔除市值或PB为负值的无效记录。

（3）行业映射：通过申万指数成分股文件（31个行业指数），将每只股票的ts_code映射到对应的一级行业。注意ts_code在文件名中为下划线格式（000001_SZ），在CSV中为点格式（000001.SZ），需进行格式统一。

（4）月度聚合：将日线数据按年-月分组，使用几何累乘法计算月度收益率：Rm = ∏(1 + rd) − 1。同月内取PE/PB/市值/换手率的月末值作为该月代表值。

（5）因子构建：每月按指定指标（市值/SMB、PB/HML、12月累计收益/MOM）将股票分为5组，取Top组与Bottom组的等权收益率之差作为因子收益率。

2.3 最终数据结构

经过预处理后，形成13个CSV文件，组织在4个层级子目录中。核心字段包括：月收益率(monthly_ret)、PE_TTM、PB、总市值(total_mv)、换手率(turnover_rate)、行业分类(l1_industry)、因子收益率(SMB/HML/MOM/Rm)及其累计值、因子拥挤度评分(crowding_score)、宏观周期阶段(cycle_phase)等。

3.2 设计原则应用

（1）颜色设计：采用金融终端暗色主题（TradingView/Bloomberg风格），以深蓝黑(#0f1923)为底色，降低视觉疲劳。核心编码规则：红涨绿跌（符合A股交易习惯），暖色=过热/危险，冷色=衰退/正常。宏观四周期分别使用蓝/红/橙/靛青区分，风格象限使用灰/橙/蓝/青四色。

（2）交互设计：所有图表均支持hover提示（显示精确数值）、图例切换（点击显示/隐藏数据系列）、缩放与平移。动态气泡图使用动画帧自动播放行业随时间迁移。综合仪表盘(index.html)提供侧边栏导航、搜索过滤和键盘快捷键（↑↓键切换图表、1-9键快速跳转）。

（3）视觉编码：气泡图中位置编码收益率-波动率二维信息，气泡大小编码市值，颜色编码行业；热力图中颜色深浅编码收益率高低；桑基图中节点颜色区分决策层级（宏观蓝/策略绿/行业橙），链接宽度编码配置强度。

3.3 工具选择

选择Python Plotly作为核心可视化工具，原因如下：（1）Plotly支持全部所需图表类型（包括Indicator仪表盘、平行坐标、桑基图等复杂图表）；（2）输出为独立HTML文件，内嵌JavaScript，无需服务器即可在浏览器中实现完整交互；（3）Kaleido引擎支持高分辨率PNG静态导出（≥300dpi），满足报告插入要求；（4）Python生态无缝衔接pandas数据处理流水线。辅助使用NetworkX（网络图布局）、Matplotlib/Seaborn（补充静态图表）。

图1宏观经济周期仪表盘

图表解读：仪表盘上半部分为PMI实时仪表，指针指向当前PMI数值，绿区(50-60)为扩张区、红区(40-50)为收缩区，直观反映经济冷暖。下半部分展示PMI、CPI、M2三条宏观指标的时间走势，背景色带标识出美林时钟四阶段。从走势图可见：2022年PMI多次下探荣枯线以下（疫情封控冲击），对应衰退期；2023年PMI反弹至扩张区间，进入复苏期；2024-2025年PMI在50附近震荡，周期切换频繁，显示经济复苏基础尚不稳固。CPI长期低于3%（温和区间），M2增速2025年后明显回升，反映宽松货币政策加码。

图2市场风格轮动热力三角图

图表解读：该图以大小盘溢价为X轴、价值成长溢价为Y轴，将风格空间划分为四个象限。每个数据点代表一个月份，气泡大小表示风格漂移强度（距原点的欧氏距离）。从图中可观察到明显的风格轮动规律：2022年数据点集中在"大盘价值"象限（市场避险情绪主导）；2023-2024年向"小盘成长"方向迁移（AI主题驱动）；2025年呈现"小盘价值"特征（小市值+低估值策略占优）。风格漂移呈现顺时针旋转模式，与宏观周期切换基本同步。

图3行业月度收益率日历热力图

图表解读：该热力图以行业为行（31个申万一级行业）、月份为列、颜色编码月收益率（红涨绿跌）。从纵向（行业维度）可识别出"行业动量效应"：某些行业连续多月领涨（如2023年的传媒、通信，受益于AI和游戏版号放开），形成明显的红色水平带。从横向（月份维度）可观察到"系统性涨跌"：2022年4月和10月出现大面积深绿色（系统性下跌），对应上海疫情封控和全球加息恐慌期。热力图还揭示了行业轮动的"冷热交替"规律——领涨行业通常持续3-6个月后切换，为行业轮动策略提供了直观的择时参考。

图4行业轮动动态气泡图

图表解读：该图以季度为动画帧，X轴为年化波动率（风险维度），Y轴为季度累计收益（回报维度），气泡大小代表总市值，颜色区分行业。播放动画可观察各行业在风险-收益平面上的迁移轨迹。一个显著的规律是：高收益行业往往伴随高波动（高风险溢价），气泡呈现右上-左下倾斜分布。2024年后，部分行业（如银行、公用事业）表现出"低波动+稳健正收益"特征（右下象限聚集），符合防御型配置偏好。动态视角清晰地展示了市场热点从"高波动高收益"向"低波动稳健收益"的结构性迁移。

图5行业相关性网络图

图表解读：力导向布局的网络图展示了31个行业之间的收益率相关性结构（仅显示相关性>0.45的边）。节点颜色从冷色到暖色表示连接度递增（即该行业与更多其他行业高度相关）。图中可识别出几个明显的"行业群落"：（1）金融周期群——银行、非银金融、房地产、建筑装饰高度互联；（2）科技成长群——电子、计算机、通信、传媒形成紧密子图；（3）大消费群——食品饮料、医药生物、家用电器聚合；（4）上游资源群——煤炭、石油石化、有色金属形成独立模块。群落内部相关性高，群落之间相关性低，这为构建分散化投资组合提供了直观依据。

图6行业多因子暴露平行坐标图

图表解读：该平行坐标图展示最新季度各行业在8个标准化因子维度（Z-score）上的暴露轮廓。每条折线代表一个行业，颜色编码其收益率Z-score。从图中可识别出几类典型的行业因子特征：（1）高成长型（电子、计算机）——PE和PB估值偏高、动量强、波动率大；（2）价值型（银行、煤炭）——PE和PB估值极低、总市值大、换手率低；（3）中小盘型——总市值偏小、波动率居中、动量分化。平行坐标的"交叉"区域表示因子特征不明显的行业，而"平行束"区域则表示一组行业的因子结构高度相似（如同属消费类的食品饮料与医药生物的折线高度重叠）。

图7因子收益分析

图表解读：左图展示市场因子(Rm)、规模因子(SMB)、价值因子(HML)、动量因子(MOM)的月度收益率柱状图，右图展示各因子的累计收益曲线。关键发现：（1）市场因子(Rm)长期累计收益为正，反映了A股整体的正风险溢价；（2）规模因子(SMB)自2023年以来累计上行，验证了小盘股相对大盘股的超额收益趋势；（3）价值因子(HML)在2022年和2025年表现突出（价值股跑赢成长股），与加息周期和避险情绪回升的时间窗口吻合；（4）动量因子(MOM)波动最大，2023年强势（趋势延续），2024年回撤（动量崩溃），提示动量策略存在周期性失效风险。

图8因子拥挤度监控仪表

图表解读：该仪表从三个维度监控因子拥挤风险：（1）估值价差——拥挤度升高时Top-Bottom组的PB价差扩张；（2）因子内相关性——拥挤导致同组个股走势趋同，组内平均相关性上升；（3）综合拥挤度分数——前两个指标的标准化均值，>2为危险、1-2为警示、-1~1为正常、<-1为关注（因子被低估）。面板3中标注了危险和警示月份。分析期内，2024年上半年出现过短暂的危险信号（小盘股过度拥挤），随后确实出现了风格切换，验证了拥挤度指标的风险预警能力。

图9牛熊周期下行业收益率分布对比

图表解读：该箱线图选取波动率最高的15个行业，按美林时钟四阶段分组展示其月度收益率分布。关键发现：（1）复苏期中位数收益率最高、箱体宽度（波动）适中，各行业普遍录得正收益；（2）过热期少数行业（有色、煤炭等资源类）收益率极高但方差也大，行业分化最严重；（3）滞胀期是最差的投资阶段——几乎所有行业中位数收益为负，传统防御型行业（公用事业）表现出相对韧性；（4）衰退期收益率分布偏负但不如滞胀期极端，部分逆周期行业（医药、必选消费）可获正收益。这一模式与经典美林时钟理论高度一致。

图10行业配置决策桑基图

图表解读：该桑基图构建了三阶段决策链路：宏观周期阶段→策略风格→推荐行业。链接宽度表示配置强度（基于历史胜率加权）。从流向图中可读出的配置逻辑示例：（1）复苏期→大盘成长→电子/计算机（AI产业周期驱动）；（2）过热期→小盘价值→有色金属/煤炭（资源品涨价周期）；（3）滞胀期→大盘价值→银行/公用事业（避险+分红策略）；（4）衰退期→大盘成长（逆周期政策预期）+小盘价值（超跌反弹）。hover提示显示每条路径的历史胜率，供投资者评估信号可靠性。

图11策略压力测试热图

图表解读：该热力图展示5种极端历史情景下31个行业的累计收益率表现：疫情封控(2022.3-5月)、市场探底(2022.9-10月)、量化风暴(2024.1-2月微盘股流动性危机)、政策组合拳(2024.9月一揽子增量政策)、关税冲击(2025.4月中美关税升级)。颜色从绿到红表示从盈利到亏损。关键发现：（1）防御型行业（公用事业、银行、交通运输）在绝大多数极端情景下表现出韧性；（2）高beta行业（电子、计算机、传媒）对正面政策冲击弹性最大（政策组合拳期间领涨），但负面冲击下也领跌；（3）行业对不同类型的冲击存在非对称反应——例如医药生物在疫情封控期间表现抗跌，但在其他冲击下与市场同步。这一分析为构建"全天候"行业配置组合提供了参考。

结论与业务建议

5.1 主要发现

  1. 宏观经济周期是行业轮动的核心驱动力。美林时钟框架在A股市场具有显著的行业配置指导价值：复苏期偏好科技成长（电子、计算机），过热期偏好上游资源（有色、煤炭），滞胀期偏好防御价值（银行、公用事业），衰退期偏好逆周期消费（医药、必选消费）。日历热力图和箱线图为这一规律提供了清晰的视觉证据。
  2. 风格因子（规模、价值、动量）对行业超额收益具有显著解释力。因子收益分析表明，规模因子(SMB)自2023年以来累计上行逾30%，小盘股系统性跑赢大盘股；价值因子(HML)表现出周期性波动，与利率环境高度相关；动量因子(MOM)波动最大且存在"动量崩溃"风险。因子拥挤度指标可提前2-3个月预警风格切换。
  3. 行业间存在稳定的群落结构。相关性网络图识别出金融周期群、科技成长群、大消费群、上游资源群四个稳定的群落。群落内部行业高度联动（同涨同跌），群落之间相关性较低。基于群落的分散化配置可有效降低组合波动。
  4. 极端情景压力测试揭示了行业韧性的非对称性。不同行业对不同类型的冲击（疫情、流动性、政策、贸易）表现出差异化的敏感度。公用事业、银行等防御型行业在多数极端情景下抗跌，而科技行业对正面政策冲击弹性最大。

5.2 业务建议

  1. 实施"宏观择时+行业轮动"策略：基于PMI和CPI信号判断当前美林时钟阶段，动态调整行业配置权重。当PMI<50且下降时（衰退期），超配防御型行业；当PMI>50且上升时（复苏期），超配成长型行业。
  2. 结合因子拥挤度进行风险控制：当拥挤度评分进入"警示"或"危险"区间时，降低相应因子暴露（如拥挤度危险时减配小盘股），等待拥挤度回归正常区间后再恢复配置。
  3. 利用行业群落构建分散化组合：确保组合在不同群落间合理分配，避免过度集中单一群落（如全部持仓集中在科技成长群），以降低系统性回撤风险。
  4. 将可视化仪表盘纳入日常投研流程：index.html综合仪表盘可作为投研团队的日常监控工具，快速浏览12个维度的市场信号，辅以预警面板的自动异常检测，提升投研效率。

5.3 局限性及改进方向

  1. 数据周期有限：当前分析仅覆盖2022–2025年共4年48个月的数据，样本量有限。某些宏观周期（如典型的"过热期"）在分析期内出现次数较少，结论的统计显著性有待更长周期数据的验证。
  2. 因子模型简化：采用Fama-French三因子（Market+SMB+HML）+动量因子的简约模型。未纳入更多潜在因子（如质量因子QMJ、低波动因子BAB等），可能遗漏了部分解释力。
  3. 行业分类粒度：申万一级行业31个分类对某些细分赛道（如半导体vs消费电子）的区分不够精细，可能掩盖子行业内部的异质性。
  4. 交互体验可进一步优化：当前图表为独立HTML文件，图表间缺乏联动（如点击热力图某行业，自动高亮网络图中该行业的节点）。未来可实现跨图表的刷选联动，提升分析深度。
