================================================================
  数据集描述 — 股票市场行业轮动与因子收益分析
  时间范围: 2022-01-01 至 2025-12-31（4 个完整年度）
  交易日数: 约 970 天（上交所 SSE 交易日历）
  更新日期: 2025-06-01

【目录结构】

  数据集/

================================================================
二、处理后数据 — processed/
================================================================

处理后数据由四层流水线脚本产出 (必须在 源代码/ 目录运行):

  python build_industry_data.py      # 第1层: 行业聚合
  python build_macro_cycle.py        # 第2层: 宏观环境
  python build_factor_attribution.py # 第3层: 因子归因
  python build_decision_support.py   # 第4层: 决策支持

每层读取上一层 CSV 输出，CSV 是层间唯一数据交换格式。

───────────────────────────────────────────────────────────────
第1层: 行业聚合 — processed/01_industry/
───────────────────────────────────────────────────────────────

    行业日度聚合指标
    字段:
