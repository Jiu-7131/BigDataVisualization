============================================================
  数据来源说明
  股票市场行业轮动与因子收益分析
============================================================

【数据源】
  Tushare Pro (https://tushare.pro)
  国内最大的免费金融数据开放平台，提供全面的A股市场数据接口。

【获取数据接口一览】
  ┌─────────────────────┬──────────────────────────┬──────────┐
  │ 接口                 │ 说明                     │ 数据量    │
  ├─────────────────────┼──────────────────────────┼──────────┤
  │ daily               │ A股日线行情 (OHLCV)       │ ~5,400只 │
  │ daily_basic         │ 每日基本面 (PE/PB/市值等) │ ~5,400只 │
  │ adj_factor          │ 复权因子                  │ ~5,400只 │
  │ index_member        │ 申万行业指数成分股        │ 31个行业 │
  │ moneyflow           │ 个股资金流向              │ ~5,400只 │
  │ style_index         │ 风格指数日线 (上证50等)   │ 5个指数  │
  │ cn_pmi              │ 中国制造业PMI             │ 月度     │
  │ cn_cpi              │ 中国居民消费价格指数      │ 月度     │
  │ cn_m                │ 中国货币供应量 (M2)       │ 月度     │
  └─────────────────────┴──────────────────────────┴──────────┘

【时间范围】
  2022年1月 — 2025年12月 (共48个月)

【原始数据规模】
  个股日线数据: 约 5,463 只股票 × 约 974 个交易日 ≈ 500万条记录
  个股基本面: 同上
  总计原始CSV文件: 约 27,300 个

【数据预处理说明】
  1. 缺失值处理: 对个别交易日的缺失PE/PB使用前向填充
  2. 异常值检测: 剔除涨跌幅超过±11%的异常记录 (A股涨跌停限制)
  3. 行业映射: 通过申万指数成分股文件将个股映射到31个一级行业
  4. 月度聚合: 将日线数据聚合为月度收益率 (几何累乘)
  5. 因子构建: 每月按市值/PB/动量五分位分组，Top-Bottom = 因子收益率

【数据可靠性】
  - Tushare Pro 数据与Wind/Choice等商用终端一致
  - 所有原始CSV文件保留在 tushare_all_interfaces/ 和 preprocessed_all_data/
  - 数据处理全过程可复现
